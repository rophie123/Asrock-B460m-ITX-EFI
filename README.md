- Audio device: front panel, back panel
- USB: USB 2 and USB 3, total 6 USB physical ports
- DP video port, audio output of DP
- Sleep
- Wake up
- iServices: iMessage, FaceTime, Apple ID, App Store, iCloud
- WiFI & Bluetooth
